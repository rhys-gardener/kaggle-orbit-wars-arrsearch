# kaggle-orbit-wars-arrsearch

Array-only search + self-play training track for the Kaggle Orbit Wars
competition (deadline 2026-06-23). A divergence from the heuristic / evolved
agents in `kaggle-orbit-wars-rg`: here, training states are generated by
synthesizing initial array states and rolling them forward with pure ndarray
operations — no kaggle env or fast-sim in the hot path.

See [docs/plans/array-search-initiative.md](docs/plans/array-search-initiative.md)
for the design and roadmap.
